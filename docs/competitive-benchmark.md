# 競合ベンチマーク

        ## 比較対象

        - Home Assistant: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- Google Home: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- Notion: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- NetBox: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。

        ## 採用する基準

        - 入力負荷が低い。
        - 次アクションが明確。
        - ローカルまたは閉域で検証できる。
        - docs と release evidence から状態を追える。

        ## 差別化

        家庭内デバイス台帳、状態、設置場所、手動点検をまとめる静的ダッシュボードに絞り、汎用ノートや大型サービスよりも閉域アルファの検証可能性を優先します。
